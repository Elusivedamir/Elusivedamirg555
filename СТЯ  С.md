# TelegramAutoBot — полная инструкция по запуску

## Что это за программа
TelegramAutoBot помогает автоматически:
- вступать в указанные каналы;
- работать со связками каналов и чатов;
- оставлять комментарии в нужных местах.

## 1. Как установить
На macOS достаточно:
1. Открыть терминал в папке проекта.
2. Выполнить:
   ```bash
   chmod +x install_full.command
   ./install_full.command
   ```
3. Программа сама:
   - установит зависимости;
   - соберёт приложение;
   - снимет quarantine;
   - установит его в Applications;
   - откроет.

## 2. Как запустить вручную
Если нужно запустить без установки:
1. Откройте терминал в папке проекта.
2. Выполните:
   ```bash
   chmod +x run_app.command
   ./run_app.command
   ```
3. Скрипт сам:
   - создаст виртуальное окружение, если его нет;
   - установит нужные пакеты;
   - запустит приложение.

## 3. Как использовать
### Полная пошаговая схема
1. Откройте программу.
2. Перейдите на вкладку "Аккаунт" и авторизуйтесь в Telegram.
3. На вкладке "Каналы" добавьте нужные каналы и нажмите "Вступить в каналы из списка".
4. На вкладке "Связки" заполните таблицу:
   - первая колонка — ID канала;
   - вторая колонка — ID чата, обсуждения или группы;
   - если нужен обычный чат, вторую колонку оставьте пустой.
5. На вкладке "Комментирование" заполните комментарий и нажмите "Запустить комментирование".
6. Следите за логом внизу окна.

### Шаг 1. Авторизация
1. Откройте программу.
2. Войдите в Telegram через номер телефона и код.
3. Если нужно, укажите прокси.

### Шаг 2. Каналы для вступления
1. Перейдите на вкладку "Каналы".
2. Добавьте каналы в список.
3. Нажмите "Вступить в каналы из списка".

### Шаг 3. Связки
Это самый важный раздел.

#### Что куда вводить
- Первая колонка: ID канала, откуда берутся публикации.
- Вторая колонка: ID чата, обсуждения или группы, куда нужно отправлять комментарий.
- Если нужен обычный чат, вторую колонку можно оставить пустой.

#### Пример
| Колонка | Что вводить |
|---|---|
| 1 | ID канала |
| 2 | ID чата/обсуждения/группы |

#### Пример значения
- Канал: -1001111111111
- Чат/обсуждение/группа: -1002222222222

### Шаг 4. Комментирование
1. Заполните текст комментария.
2. Нажмите "Запустить комментирование".
3. Если нужно — остановите процесс кнопкой "Остановить".

## 3. Где брать ID
- ID канала можно получить из Telegram через бота или парсер.
- ID чата/обсуждения/группы указывается во второй колонке.
- Для супергруппы или обсуждения обычно используется ID вида -1001234567890.

## 4. Если что-то не работает
- Проверьте авторизацию.
- Убедитесь, что список каналов не пуст.
- Убедитесь, что файл связок заполнен правильно.
- Если приложение не запускается, повторно выполните install_full.command.
